Portfolio Website

Links: GitHub

This website was made with only HTML, CSS, and client-side JavaScript as an exercise to build the website from the ground up. It also meant that I could use GitHub pages to host the website for free.

I intended to make a website that would look and feel modern, but be very lightweight and responsive. This meant that for media like the images and "GIFs" shown in the other pages I opted for the webP format is it allowed for a smaller file size while retaining much more quality as compared to other formats like PNG or GIF.

HTML was used to set the layout and the content of the website, and then used CSS to apply a modern look and feel. For some of the interactability like the filter buttons client-side JavaScript was used to improve the function of the website, allowing a visitor to specify which projects they want to see.
